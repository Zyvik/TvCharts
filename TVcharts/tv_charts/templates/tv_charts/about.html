About this project

On this site you can see how much your favourite TV show dropped in quality over time.

I've scraped data (from IMDB) for over 1000 most popular shows (nearly 100k episodes) and presented it in form of graphs.
In order to do so I created very simple API (you can check it out here)
On my github you can find both scraper that I've written and this project itself.
